## Install

### Install dependencies
```
$ npm install / npm i
```

### Run dev
```
$ npm run start
```

### Prod build
```
$ npm run build
```

### Run cypress tests
```
$ npm run test:cypress
```

### Run int tests
```
$ npm run test
```
